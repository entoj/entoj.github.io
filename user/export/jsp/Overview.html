JSP

JSP exporter

A jsp artefact exporter. All configured entities will export their templates as single jsp files. Macro calls are resolved at runtime via jsp:include.

Feature Support

Yield macros

Macros calls with custom body content like

{% call macro() %}
    <p>STUFF</p>
{% endcall %}

are supported by automatically inlining the called macro.
