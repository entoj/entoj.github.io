# Installation based on a blueprint

{% method %}
Create a local directory for your project
{% sample lang="js" %}
```bash
$ mkdir my-project
$ cd my-project
```
{% endmethod %}

{% method %}
Clone the blueprint
{% sample lang="js" %}
```bash
$ git clone https://github.com/entoj/entoj-blueprint-sass-jspm .
```
{% endmethod %}

{% method %}
Install all dependencies via npm
{% sample lang="js" %}
```bash
$ npm install
```
{% endmethod %}
